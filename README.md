# тг бот поддержки банка 
### описание
небольшой проект для хакатона на python.

бот отвечает только на банковские вопросы, информацию берет из базы данных 

бот: @UNIBANK_supportbot

### стек
python 3.11
telebot (простая библиотека для написания тг ботов)

GigaChat API (бесплатная LLM)

chromadb (векторная база данных)

### Алгоритм работы:
- пользователь отправляет вопрос в бота
- вопрос ищется базе при помощи семантического поиска (косинусного сходства)
- в gigachat отправляется сообщение пользователя, контекст из базы данных и системное сообщение
- ответ гигачата отправляется пользователю
